General Info

This program can be used to remove unused Credit Photos, Scene Photos and Cover Images.

If you wish to clean up your Cover Images folder, the set the checkbox and select the folder that contains your database.

Table of Contents

Meaning of the Individual Tabs

Credit Photos

In DVD Profiler (under Tools -> Options -> File Folders) you can set a folder where you store cast and crew pictures.
If these files follow the correct naming conventions, DVD Profiler will show these pictures when you select a cats or crew member.

In addition to the general use of credit photos you can also store them by a specific profile (e.g. if you have an actor that was still a child in one movie and an adult in a later one).
You can create a folder with the ID of that profile and put the credit photos in there.
The naming convention is explained here but only contains the parts identifier[.locality].

Scene Photos

In DVD Profiler (under Tools -> Options -> File Folders) you can set a folder where you store scene pictures.
These files are stored in one folder per profile (see naming conventions of folders here).
You can watch these pictures in DVD Profiler when you activate the gallery (View -> Windows -> Gallery).

Cover Images

Sometimes when profiles are removed from DVD Profiler the cover images remain (this is an option when you remove multiple profiles at once).
DVD Profiler already gives the option to clean up the cover images folder (under Tools -> Options -> Utilities -> Clean Images Directory).
That's why this option is disabled by default. If you wish to use it, set the check box and select your database folder.
The naming convention of the files is explained here.
